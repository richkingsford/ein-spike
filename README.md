# ein-spike

A spike to discover free or low-cost ways to validate EIN numbers for non-profits.

## Goal
Explore available APIs, databases, and methods to verify that an EIN belongs to a valid 501(c)(3) or other non-profit organization.

## Resources to Investigate
- IRS Tax Exempt Organization Search API
- ProPublica Nonprofit Explorer API
- Open990 data
- GuideStar / Candid API
